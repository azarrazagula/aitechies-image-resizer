# 📸 AiTechies Social Media Image Resizer

> A beautiful, modern, and super-fast tool to resize your images for all social media platforms.
>
> 🌐 **Live Website Link**: [https://aitechies-image-resizer.netlify.app/](https://aitechies-image-resizer.netlify.app/)

---

## 👶 What is this project? (Explained Simply)
Imagine you have an image and you want to post it on Instagram, LinkedIn, or YouTube, but the size is wrong. 

This tool helps you **resize any image** to fit perfectly on any platform in just 1 click! 

🔒 **100% Private**: Your images never leave your computer. Everything happens inside your browser!

---

## 🛠️ What technology did we use?
We used modern and powerful tools to make this app fast and beautiful:
1. **Next.js & React**: The foundation of the app.
2. **TypeScript**: Keeps our code clean and prevents errors.
3. **Tailwind CSS**: Makes our website look beautiful with a premium dark theme.
4. **Framer Motion**: Adds smooth, satisfying animations (like when you drag a file or click buttons).
5. **JSZip**: Packs multiple resized images into a single `.zip` file for quick download.

- Created a beautiful dark-themed dashboard.

## 💻 How to run the project on your computer?

Follow these simple steps:

### Step 1: Install Dependencies
Open your terminal inside the project folder and run:
```bash
npm install
```

### Step 2: Run the Development Server
Start the website locally by running:
```bash
npm run dev
```

### Step 3: Open in Browser
Open your browser and visit:
[http://localhost:3000](http://localhost:3000)

Now you can upload images, resize them, and download them! 🎉
